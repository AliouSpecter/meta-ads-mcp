# 10 Prompts Meta Ads MCP — Operationnels

> A utiliser directement dans Claude une fois le MCP Meta connecte.

---

## Audit & Diagnostic

**1. Bilan compte du matin**
```
Donne-moi un resume des performances de mes campagnes Meta des 7 derniers jours : depenses totales, ROAS moyen, CPM, CTR. Signale ce qui a baisse de plus de 20%.
```

**2. Audit qualite Pixel**
```
Verifie la qualite du signal de mon Pixel Meta. Quels evenements sont bien recus ? Lesquels ont des problemes de deduplication ou de signal faible ?
```

**3. Detection budget en derive**
```
Parmi mes campagnes actives, lesquelles ont depasse leur budget quotidien ou sont en sous-livraison ? Donne-moi les details et une recommandation.
```

**4. Analyse fatigue creative**
```
Quelles sont mes creatives actives depuis plus de 3 semaines avec une frequence superieure a 3 ? Liste-les avec leur CTR et ROAS actuels.
```

---

## Creation & Optimisation

**5. Creation campagne complete**
```
Cree une campagne Meta Ads pour [produit/service]. Objectif : [conversions/leads/trafic]. Budget quotidien : [X]€. Audience cible : [description]. Propose le ciblage, les placements et 3 variantes de copy.
```

**6. Generation de variantes copy**
```
Prends ma campagne [nom] et genere 5 variantes de primary text pour les tester en A/B. Garde le meme angle mais varie le hook et le CTA.
```

**7. Optimisation audience**
```
Analyse les ad sets de ma campagne [nom]. Quelles audiences performent le mieux sur les 14 derniers jours ? Lesquelles faut-il couper ou scaler ?
```

---

## Reporting

**8. Rapport hebdomadaire**
```
Genere un rapport de performance de la semaine passee pour tous mes comptes actifs. Format : depenses, ROAS, CPL, impressions, et top 3 creatives par compte.
```

**9. Comparaison periode**
```
Compare les performances de mes campagnes ce mois-ci vs le mois dernier. Quels sont les 3 changements les plus significatifs ?
```

**10. Audit catalogue produit**
```
Analyse mon catalogue produit Meta. Y a-t-il des produits rejetes, des erreurs de flux ou des articles avec un taux de clics anormalement bas ?
```
